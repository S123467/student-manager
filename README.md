# 学生信息管理系统

## 项目简介

这是一个基于 **C++ + Qt5** 开发的学生信息管理系统，用于大学生计算机设计大赛参赛项目。
该软件实现了学生信息的增删改查、文件存储等完整功能，界面简洁清晰，代码结构规范，适合比赛提交和答辩。

## 功能特性

- ✅ **添加学生**：录入新的学生信息（学号、姓名、性别、年龄、专业、绩点）
- ✅ **删除学生**：删除已添加的学生信息，带有确认提示
- ✅ **修改更新**：更新已存在学生的信息
- ✅ **查询搜索**：支持按姓名关键字模糊搜索
- ✅ **显示全部**：查看所有学生信息
- ✅ **文件保存**：将数据保存到本地文件
- ✅ **文件加载**：从本地文件加载数据
- ✅ **图形界面**：直观易用的GUI界面，表格展示

## 技术栈

- **编程语言**：C++ (C++11标准)
- **GUI框架**：Qt 5.x
- **构建工具**：CMake
- **数据存储**：纯文本文件，简单可靠

## 类设计说明

| 类名 | 职责说明 |
|------|----------|
| Student | 存储单个学生的基本信息，提供getter/setter方法 |
| DataManager | 管理所有学生数据，实现增删改查算法和文件读写 |
| MainWindow | 创建GUI界面，处理用户交互，调用DataManager |

## 编译运行

### 环境要求

- 安装 Qt 5.x 开发库
- 安装 CMake (版本 3.5 以上)
- 支持 C++11 的编译器 (GCC 4.8+ / Clang / MSVC)

### 编译步骤

```bash
# 进入项目目录
cd student-manager

# 创建构建目录
mkdir build && cd build

# 生成Makefile
cmake ..

# 编译
make -j4

# 运行
./StudentManager
```

### Ubuntu/Debian 环境安装依赖

```bash
sudo apt-get update
sudo apt-get install qtbase5-dev cmake g++
```

## 使用说明

1. **添加学生**：填写所有信息后点击「添加」按钮
2. **选择学生**：点击表格中的某一行，信息会自动填充到输入框
3. **更新信息**：选择学生后修改信息，点击「更新」按钮保存修改
4. **删除学生**：选择学生后点击「删除」按钮，确认后删除
5. **搜索学生**：在搜索框输入姓名关键字，点击「搜索」
6. **显示全部**：点击「显示全部」显示所有学生
7. **清空输入**：点击「清空」清空所有输入框
8. **保存文件**：点击「保存到文件」将数据保存到文件
9. **加载文件**：点击「从文件加载」从文件读取数据
